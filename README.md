# Base Template

Configures `.gitignore` and GitHub Actions CI for a Rails 8 app with PostgreSQL. This is the recommended first template to apply — it sets up the git hygiene and CI pipeline that other templates (particularly `rspec`) build on.

## What It Does

1. Rewrites `config/database.yml` with `host: localhost`, `username: postgres`, `password: postgres` for development and test environments; production uses `ENV` variables. Runs `db:create`.
   - Update these credentials if your PostgreSQL setup differs

2. Appends `/vendor/bundle` to `.gitignore`
   - Prevents bundled gems from being committed to version control
   - Reminder to run `bundle install` after cloning

3. Writes `.github/workflows/ci.yml` with a full GitHub Actions pipeline:
   - **scan_ruby** — Brakeman static analysis for Rails security vulnerabilities
   - **scan_js** — importmap audit for JavaScript dependency vulnerabilities
   - **lint** — RuboCop with GitHub annotations format
   - **test** — Minitest and system tests with a PostgreSQL service container; screenshots from failed system tests are uploaded as artifacts

## Dependencies

- Rails 8 with PostgreSQL (`rails new my-app -d postgresql`)
- A `.ruby-version` file in the project root (generated by `rails new`)

## Usage

**Option 1 — Raw URL**

Navigate to this branch, open `template.rb`, click **Raw**, and pass the URL.

**Option 2 — Local file**

Copy `template.rb` into your project directory and pass the local path instead.

### New app

```bash
rails new my-app -d postgresql -c tailwind -m https://raw.githubusercontent.com/rogue-media-lab/rails-templates/refs/heads/base/template.rb
```

### Existing app

```bash
bin/rails app:template LOCATION=https://raw.githubusercontent.com/rogue-media-lab/rails-templates/refs/heads/base/template.rb
```

## Compatibility with RSpec

The [`rspec` template](../../tree/rspec) detects whether `.github/workflows/ci.yml` exists and, if so, replaces the Minitest test command with `bundle exec rspec`. Apply `base` before `rspec` to get CI updated automatically.
